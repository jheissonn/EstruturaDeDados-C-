<p>This Java program will validate and solve Sudoku. 
If you have a Sudoku problem on your hands that you cannot solve, 
you can type those numbers in and let the program solve.</p>
<hr />
*How to use*
<ul>
<li>Arrow keys: navigate through cells</li>
<li>1-9 keys: input</li>
<li>Backspace key: erase your input</li>
</ul>

<ul>
<li>Submit button: Checks whether the Sudoku is complete or not.</li>
<li>Solve button: Generates a solution from current state.</li>
<li>Erase button: Erases all cells except preset cells.</li>
<li>Erase All button: Erases all of the cells.</li>
<li>Preset button: Make filled cells fixed.</li>
</ul>
